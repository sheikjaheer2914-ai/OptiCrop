# Demonstration of Proposed Solution Features

Our team live demo will walk through the two core interactive application modes built inside the dashboard:

## Feature Module 1: Smart Crop Selector Form
- **Interaction Walkthrough:** Showcases entering standard on-field N, P, K values along with local monsoon rainfall levels.
- **System Action:** Shows the backend normalizer transforming the input data vector and running the classification loop to dynamically display the optimal seed type banner.

## Feature Module 2: Field Compatibility Diagnostic Matrix
- **Interaction Walkthrough:** Demonstrates typing in an individual specific crop name (e.g., maize) and clicking the evaluation button.
- **System Action:** Displays the internal class match probability percentage and renders a green, yellow, or red risk flag box for the user.
