# Scalability and Future Roadmap

## 1. System Engineering Limitations
The current standalone prototype relies entirely on manually entered slider fields and operates on a localized dataset.

## 2. Future Development Scope (Phase 2 Expansion)
- **Live Weather API Integration:** Hook up open-source regional weather services to pull real-time temperature, humidity, and rainfall logs based on the user's GPS coordinates automatically.
- **Deep Learning Image Diagnostics:** Introduce a secondary computer vision tracking tab using Convolutional Neural Networks (CNNs) to diagnose leaf diseases from uploaded field photos.
- **Cloud Database Clusters:** Migrate flat serialized binary files () into an online API hosting grid to support hundreds of parallel user queries.
