# Brainstorming & Idea Prioritization Matrix

## 1. Brainstormed Project Feature Concepts
- **Concept A:** Real-time multi-class crop recommendation based on local parameters (High Feasibility / High Impact).
- **Concept B:** Automated field drone navigation tracking loops (Low Feasibility / High Cost).
- **Concept C:** Soil NPK nutrient deficiency balancing optimizer (High Feasibility / Medium Impact).

## 2. Prioritization Framework

| Feature Concept | Technical Feasibility | Expected User Impact | Priority Rank |
| :--- | :--- | :--- | :--- |
| **Crop Recommendation Engine** | High | High | **Priority 1** |
| **Soil Nutrient Adjuster** | High | Medium | **Priority 2** |
| **Drone Tracking Loop** | Low | High | **Priority 3** |

## 3. Core Project Scope Selected
Our group selected **Priority 1 (Crop Recommendation)** and **Priority 2 (Soil Nutrient Adjuster)** as our structural core milestones to guarantee a reliable machine learning validation flow within our project deadline.
