# Technology Stack Specification: OptiCrop Engine

## 1. Core Development Language
- **Python 3.10+**: Main programming language for building the analytics pipelines, training algorithms, and operating the web server interface.

## 2. Machine Learning & Mathematical Infrastructure
- **Pandas & NumPy**: For structural matrix array formatting, cleansing null cells, and feature selection manipulation.
- **Scikit-Learn**: Powering our core multi-class algorithm library (Random Forest Classifier).
- **Joblib**: Used to unpickle serialized binaries () to execute production predictions.

## 3. Front-End User Interface & Cloud Operations
- **Streamlit**: Web server application framework used to deploy reactive interactive forms, numeric data inputs, and graphical reporting layout nodes.
- **Git & GitHub Platform**: Framework utilized to manage branch changes, commit track histories, and link project files directly to our Skill Wallet dashboard.
