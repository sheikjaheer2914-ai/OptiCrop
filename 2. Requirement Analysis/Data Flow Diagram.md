## Level 1: Detailed Process Flow
1. **User Input Process:** Farmer feeds 7 environmental parameters via the browser forms.
2. **Data Transformation Process:** Inputs are structured into a NumPy vector array matrix and scaled using standard normal distribution weights ().
3. **Inference Execution Process:** Normalized array maps through trained Random Forest Decision Tree trees.
4. **Output Rendering Process:** High-probability target crop classifications match up with localized nutrient advisory text on the UI.
